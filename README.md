# Component Framework 

> Generic, reusable front-end component framework. 

> Named after your friend and mine, [Andrew Blackledge](https://twitter.com/andy_blackledge).

## Usage
```sh
git clone git@github.com:furzeface/blackledge.git
cd blackledge
npm install
grunt serve
```
